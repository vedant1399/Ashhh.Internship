# Internship Project - All Tasks

This repo contains the 7 task folders for the internship assignment:
- Task 1 - Backend Setup & Core Models
- Task 2 - User Auth API (Register-Login-Logout)
- Task 3 - Project CRUD API
- Task 4 - Task CRUD API
- Task 5 - Frontend (React - Vite/Axios)
- Task 6 - Filtering & Sorting Tasks
- Task 7 - API Documentation + Testing (Postman Collection)

Each folder contains code snippets, README and example files.
The React frontend zip for Task 5 is included as `react-task-manager.zip`.
